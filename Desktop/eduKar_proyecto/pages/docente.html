Proyectos Educativos

¿Qué es un proyecto educativo?

Es un plan para que busca alcanzar una meta de caracter formativo. Es decir se trata de planificar un proceso que surge a partir de un diagnóstico de una necesidad con el fin de promover un aprendizaje y de resolver una determinada problemática.
